# Weather Information App (Java Swing)

## Requirements:
- Java 11 or higher
- Internet connection
- org.json library included in /lib

## How to Run:
1. Extract the ZIP file.
2. Open the project in any Java IDE (VS Code, IntelliJ, NetBeans).
3. Ensure the 'lib/json-20210307.jar' is added to the classpath.
4. Run app.WeatherAppSwing.

Notes:
- Background images are located in /resources/backgrounds
- Replace YOUR_API_KEY in WeatherService.java before running.
